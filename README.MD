# DisnakePromoBot Discord Bot
<div align="center">

Этот проект представляет собой бота для Discord, написанного на Python с использованием библиотеки **Disnake**. Бот позволяет создавать промкоды, а также хранит данные в базе данных **SQLite**.


---

## Описание

Бот предоставляет следующие возможности: создавать промокоды, посмотреть промокоды, активировать промокоды




### Команды:
/promo - активировать промокод
/viewspromo - посмотреть активные промокоды
/createpromo - создать промокод
